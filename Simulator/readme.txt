How to use CAN-BT Simulator WiFi version
----

0. Preparation
- Install JAVA 6 to your Mac(PC) to run CAN-BT simulator WiFi version.
- When you use real iPhone (iOS device) to test your application,
  connect iPhone and Mac (CAN-BT simulator) in same WiFi network.
  See 'test_image.ppt' file.


1. Use CAN-BT Simulator
- run 'simulator.jar'.
- click 'listen' button to wait for connection from your application (iPhone or its simulator)
- run your application on iPhone (iOS device) and connect to CAN-BT simulator.
